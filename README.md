# Dungerouns
Metroidvania feito no Game Maker Studio 1.4 em 2019.

## Instruções
Clique: interação com o cenário
A e S: para mover
W: pula
S: se agacha
Shfit: corre
Shift+S: rasteira

## Modo Debug
Ctrl+R: reinicia o jogo
Ctrl+K: reinicia a room
Ctrl+N: room anterior
Ctrl+M: próxima room
Ctrl+X: encerra o jogo
